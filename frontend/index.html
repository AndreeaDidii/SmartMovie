  • 🎥 TMDB Integration – Get movie posters, actors, genres, and more from a real-time API.
  • 🧠 LLaMA 3 Reasoning – See why a movie is right for you, explained in natural language.
  • 💾 Local Database – Fast data retrieval using an optimized SQLite backend.

About This Project

This project started from a simple frustration — spending more time searching for a movie than actually watching one. I wanted to change that.

With a deep love for cinema and a background in data, I set out to build something smarter: a system that doesn't just list popular titles but actually tries to understand your preferences. A tool that feels personal, not generic.

Every detail here — from how recommendations are made to the way results are presented — was created with the user experience in mind. If it helps you discover a new favorite film or saves you a few indecisive hours, then it’s done its job.
